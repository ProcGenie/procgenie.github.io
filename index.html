ProcGenie

An Open Source Tool for Hand Authored Procedural Text Generation
Using Replacement Grammars

  • Open source visual authoring tool for handcrafted, coherent, and interactive procedural text generation

  • Browser based user interface for defining replacement grammars on graphs

  • Constraint-based, stateful, and probabilistic approach to procedural text generation

  • Procedurally generate interactive stories or steer generation with dynamically created choices, hyperlinks, and parser actions

  • Process text mid- and post-generation using a suite of text processing and parsing functions
